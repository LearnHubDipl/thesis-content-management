    \chapter*{Abstract}
\end{spacing}
\begin{wrapfigure}{r}{0.3\textwidth}
    \begin{center}
      \includegraphics[width=0.2\textwidth]{pics/logo_learnhub.png}
    \end{center}
\end{wrapfigure}
This thesis presents LearnHub, a web-based content management platform developed for students and teachers at HTL Leonding to support structured Matura preparation. Existing learning platforms such as StudyDrive, Knowunity, and StudySmarter were found to lack specialization for HTL-specific subjects, contain advertising, or require paid subscriptions.
LearnHub addresses these shortcomings by providing three core modules: a Matura Upload system for organizing class notes and learning materials by subject and topic pool, a Question Configurator for creating and managing practice questions with a teacher-controlled approval workflow, and a User Management system based on the school's existing LDAP infrastructure for role-based access control.
The platform was built using Angular for the frontend, Quarkus as the Java-based backend, and containerized via Docker and Docker Compose for deployment on a school-owned virtual machine. Authentication is handled through Keycloak and the HTL's LDAP directory, ensuring seamless login for all school members without separate registration.
LearnHub was developed collaboratively with a partner team, whose complementary diploma thesis implements the Matura Trainer component for practicing and simulating exams.

\begin{spacing}{1}
    \chapter*{Zusammenfassung}
\end{spacing}
\begin{wrapfigure}{r}{0.3\textwidth}
    \begin{center}
      \includegraphics[width=0.2\textwidth]{pics/logo_learnhub.png}

Diese Diplomarbeit beschreibt die Entwicklung von LearnHub, einer webbasierten Content-Management-Plattform für Schüler:innen und Lehrer:innen der HTL Leonding zur strukturierten Maturavorbereitung. Eine Marktanalyse bestehender Lernplattformen – darunter StudyDrive, Knowunity und StudySmarter – zeigte, dass keine dieser Lösungen auf HTL-spezifische Fächer zugeschnitten ist oder einen qualitätsgesicherten, werbefreien Zugang ohne Premium-Zwang bietet.
LearnHub schließt diese Lücke mit drei zentralen Modulen: dem Maturaupload zum strukturierten Ablegen und Abrufen von Mitschriften nach Fächern und Themenpools, dem Fragenkonfigurator zum Erstellen und Üben von Maturafragen mit einem lehrerseitigen Freigabeprozess sowie dem User Management für eine rollenbasierte Zugriffskontrolle auf Basis des schuleigenen LDAP-Systems.
Das Frontend wurde mit Angular, das Backend mit Quarkus umgesetzt. Die Anwendung wird mittels Docker und Docker Compose containerisiert und auf einer schuleigenen virtuellen Maschine betrieben. Die Authentifizierung erfolgt über Keycloak in Verbindung mit dem HTL-LDAP-Verzeichnis, sodass keine separate Registrierung notwendig ist.
LearnHub entstand in Zusammenarbeit mit einem Partnerteam, das in einer eigenständigen Diplomarbeit den Maturatrainer – eine Komponente zum Lernen und Simulieren von Prüfungen – entwickelte.
