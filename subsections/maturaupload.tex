\subsection{Strukturierung der Fächer und Themenpools}
Ein Klick auf „Maturavorbereitung“ in der Navigationsleiste führt die User zur Auswahl der Fächer.

\begin{figure}[h t]
    \centering
    \includegraphics[width=0.6\textwidth]{pics/ab1.png}
    \caption{Aufbau der Fächer}
\end{figure}

Das Fächer- und Themenpool-System ist die Art und Weise, wie die Datensätze in der Matura Vorbereitungsplattform organisiert sind.
Dieses System ermöglicht eine eindeutige Kategorisierung von Fragen und Materialien nach Bereichen und vereinfacht so die Navigation und Verwaltung.

\subsubsection{Fächer}
\begin{figure}[h t]
    \centering
    \includegraphics[width=0.6\textwidth]{pics/ab2.png}
    \caption{Upload der Fächer- und Themenstruktur}
\end{figure}

\begin{itemize}
    \item Jedes Fach (z.\,B. \textit{Mathematik}, \textit{Englisch}, \textit{Deutsch}) kann von Usern erstellt, bearbeitet und gelöscht werden.
    \item Ein Fach ist die oberste Ebene dieser Struktur und enthält eine oder mehrere Themenpools.
    \item Bei der Erstellung eines Faches werden Fachname, Beschreibung, Bild-URL und die zugehörigen Themenpools angegeben.
\end{itemize}

\subsubsection{Themenpools}
\begin{figure}[h t]
    \centering
    \includegraphics[width=0.6\textwidth]{pics/ab3.png}
    \caption{Themenpools innerhalb eines Faches}
\end{figure}

\begin{itemize}
    \item Themenpools sind inhaltliche Unterkategorien innerhalb eines Faches, z.\,B. \textit{Qualitätssicherung} oder \textit{Kontrolle und Steuerung}.
    \item Jeder Themenpool ist einem Fach zugeordnet und beinhaltet die zugehörigen Unterlagen. 
    \item Ebenso wie Fächer können Themenpools erstellt, bearbeitet und gelöscht werden. 
    \item Wenn in der Fachauswahl das Fach gewechselt wird, aktualisieren sich die zugehörigen Themenpools automatisch.
\end{itemize}

\subsubsection{Verknüpfung mit dem Fragenkonfigurator}
\begin{itemize}
    \item Jede Frage wird damit eindeutig einem Fach und Themenpool zugeordnet, sodass im Lernmodus danach filtert werden kann. 
    \item Dadurch wird eine sehr konsistente Organisation der Inhalte sowie eine sehr gute Navigation aller Inhalte ermöglicht.
\end{itemize}

\subsection{Upload und Verwaltung von Unterlagen}
\begin{figure}[h t]
    \centering
    \includegraphics[width=0.6\textwidth]{pics/ab4.png}
    \caption{Upload und Verwaltung von Unterlagen}
\end{figure}

Die Plattform ermöglicht das Hochladen und Verwalten weiterer Lernmaterialien, wie beispielsweise Skripte, Zusammenfassungen oder Beispielaufgaben.
Der Lernprozess wird somit durch begleitende Materialien verlängert.

\subsubsection{Upload}

\begin{figure}[H]
    \centering
    \includegraphics[width=0.6\textwidth]{pics/ab5.png}
    \caption{Upload-Ansicht für Unterlagen}
    \label{fig:upload-ansicht}
\end{figure}

\begin{itemize}
    \item n der Upload-Ansicht können Dateien im pdf – Format hochgeladen werden.
    \item Des Weiteren kann der User das Fach und Themenpool auswählen, zu dem das Dokument gehört.
    \item Bevor der User die Datei hochlädt, müssen sie Titel, Beschreibung und die zuständige Lehrkraft eintragen.
\end{itemize}

\subsubsection{Verwaltung}
\begin{figure}[h t]
    \centering
    \includegraphics[width=0.6\textwidth]{pics/ab6.png}
    \caption{Mitschriften-Übersicht}
\end{figure}

\begin{itemize}
    \item Admins können alle hochgeladenen Unterlagen ansehen, bearbeiten oder löschen.
    \item Schüler:innen haben ausschließlich Leserechte und können die Unterlagen herunterladen oder direkt im Browser anzeigen.
\end{itemize}
